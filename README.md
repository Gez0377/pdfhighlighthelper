# PDF 高亮颜色修改助手

预编译版本下载：https://pan.baidu.com/s/1hrBaLs4

使用Autohotkey模拟手动修改颜色操作，缩短高亮颜色修改时间。

Adobe Reader DC 中文版测试通过。

使用说明：
1. 只能作用于已经高亮的部分（可以使用“键盘或鼠标右键+h”快速标记高亮）。
3. 使用Ctrl+1~5，修改颜色。
5. 程序的DPI缩放可能会影响使用。
