# PDF 高亮颜色修改助手

使用Autohotkey模拟**手动修改颜色**操作，缩短高亮颜色修改时间。

Pdf xchange viewer在==1366\*768==的分辨率下中文版测试通过。
:exlamation: 只有在该特定的分辨率下才有效

使用说明：
1. 只能作用于已经高亮的部分（可以使用“键盘或鼠标右键+h”快速标记高亮）。
3. 使用Ctrl+1~5，修改颜色。
5. 程序的DPI缩放可能会影响使用。
